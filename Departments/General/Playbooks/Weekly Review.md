# Weekly Review

> Run every Friday afternoon to close out the week and plan the next. ~30 minutes.

## When to use

End of every working week. Block time on the calendar.

## Steps

1. **Tasks** — open Notion `My Tasks` view. Mark anything completed as Done. For carry-overs, push the `Due` date or move to next week's plan. Archive anything no longer relevant.
2. **Projects** — open the active Projects view per department. Update Status on anything that progressed or stalled. Note blockers in `Summary`.
3. **Decisions** — log any meaningful decisions you made this week into the Decisions Log DB.
4. **Customer Feedback** — quick triage of any new themes from HubSpot / direct messages. Add to Customer Feedback DB if recurring.
5. **Next week** — pick top 3 priorities for next week. Create or pin tasks accordingly.
6. **Note wins + lessons** — add a short note to the Decisions Log or weekly journal: what worked, what didn't.

## Notes / gotchas

- Don't try to clean up everything in one session — focus on triaging the queue, not finishing it.
- If a task has been "carrying over" for 3+ weeks, decide: do it now, delete it, or move to a long-horizon project.
